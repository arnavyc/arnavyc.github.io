    This post shows how to remove Microsoft Store apps form User Account and the System. Provisioned Apps Get list of provisioned Microsoft Store apps To get list of all provisioned apps, run the following command in PowerShell (or PowerShell Core): $ Get-ProvisionedAppxPackage -Online Remove provisioned apps you don’t require To remove unwanted provisioned apps, run the following command in an elevated PowerShell (or PowerShell Core): $ Remove-ProvisionedAppxPackage -Online -PackageName <Package Name> replacing <Package Name> with the name of package you want to remove.…
